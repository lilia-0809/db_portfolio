# 📁 Project Portfolio Bot

Telegram бот для управления портфолио проектов. Помогает сохранять информацию о проектах, навыках и статусах разработки.

## 🚀 Функционал

- ➕ **Добавление проектов** - создание новых проектов с названием, ссылкой и статусом
- 🛠 **Управление навыками** - добавление навыков к проектам
- 📁 **Просмотр проектов** - список всех проектов с возможностью просмотра деталей
- 🗑 **Удаление проектов** - удаление проектов из базы данных
- ✏️ **Обновление информации** - изменение названия, описания, ссылки или статуса проекта

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Запуск бота и приветствие |
| `/info` | Список доступных команд |
| `/new_project` | Добавление нового проекта |
| `/skills` | Добавление навыков к проекту |
| `/projects` | Показывает список проектов |
| `/delete` | Удаление проекта |
| `/update_projects` | Обновление информации о проекте |

## 🛠️ Технологии

- Python 3.12
- SQLite3 - база данных
- pyTelegramBotAPI (telebot) - Telegram Bot API

## 📁 Структура проекта

```
.
├── bot.py          # Основной файл бота с обработчиками команд
├── logic.py        # Класс DB_Manager для работы с базой данных
├── config.py       # Конфигурация (TOKEN, DATABASE)
└── README.md       # Документация
```

## 🗄️ База данных

Бот использует SQLite с следующими таблицами:

- **projects** - информация о проектах
- **skills** - список навыков
- **project_skills** - связь проектов и навыков
- **status** - статусы проектов

## 🚀 Установка

1. Установите зависимости:
```bash
pip install pyTelegramBotAPI
```

2. Добавьте токен вашего бота в файл `config.py`:
```python
TOKEN = 'ваш_токен_от_BotFather'
```

3. Запустите бота:
```bash
python bot.py
```

## 💡 Использование

1. Запустите бота командой `/start`
2. Добавьте проект с помощью `/new_project`
3. Добавьте навыки к проекту через `/skills`
4. Просматривайте проекты командой `/projects`
5. Обновляйте информацию через `/update_projects`
